# 🗺️ EONET - Natural Event Tracker

Website: [eonet.reubenwilliam.xyz](https://eonet.reubenwilliam.xyz)

Real-time global natural event tracker built using Next.js, React leaflet and NASA's EONET API.

## Features

- **Real-time data** - information about natural events occurring across the world from NASA
- **Events filtering** - filtering events by category, source, magnitude, etc.
- **Category key** - color coded key showing categories differentiated by color
- **Map controls** - basic map controls like zoom in/out, reset and map popups
